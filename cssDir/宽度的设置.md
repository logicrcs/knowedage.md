# 宽度的设置

- 直接使用px进行设置

  ```css
  <!--但这样就只能写死-->
  width:5px
  ```

- 使用百分比进行设置

  ```css
  /**
  * 能够实现比较好的自定义，但是如果父级
  * 没有宽度定义，比如一个固定的宽度值
  * 就会出现各种各样的问题
  */
  width:100%
  ```

- 使用vw进行设置

  ```css
  /**
  * 使用与移动端，不需要父级元素的宽度定义
  * 也是将总宽度分解成100分，使用上也和百分比
  * 设置很接近
  */
  width:100vw
  ```
