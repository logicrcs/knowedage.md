# css中的常用伪类

- ：active

  应用于被激活的元素，如被点击的链接、被按下的按钮
  
- :last-of-type

  代表父元素下最后一个指定类型的元素

  ```html
  <div class = "test"></div>
  <div class = "test"></div>
  <div class = "test"></div>
  <div class = "test"></div>
  <div class = "test"></div>
  <div class = "test"></div>
  ```

  ```css
  .test:last-of-type{
    
  }
  ```
